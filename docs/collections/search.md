# Cerca collezioni

<img src="../../images/sections/collections/search.png" alt="Cerca collezioni" style="box-shadow: 0 .25rem .75rem rgba(0,0,0,.05); border: 1px solid #E9EBEE; border-radius: 0.3px;"/>

Nel menù Servizi della sezione Collezioni è possibile selezionare lo strumento [Cerca collezioni](/collections/search).
Nella sezione [Cerca collezioni](/collections/search)  è possibile cercare una collezione utilizzando i descrittori disponibili.

Il primo campo *(Cerca una collezione)* consente di cercare in base alle parole contenute nel Titolo della collezione. È possibile utilizzarlo congiuntamente agli altri descrittori.

Gli altri parametri sono descritti nella sezione **[Crea collezioni](/collections/create)**.
