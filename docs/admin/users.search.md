# Cerca utenti

<img src="../../images/sections/admin/users.search/search_bar.png" alt="Barra di ricerca degli utenti" style="box-shadow: 0 .25rem .75rem rgba(0,0,0,.05); border: 1px solid #E9EBEE; border-radius: 0.3px;"/>

Nel menù Servizi della sezione Admin è possibile selezionare lo strumento [Cerca utenti](/admin/users.search).
Con questo strumento è possibile cercare un utente iscritto alla piattaforma.

Digitando nella barra di ricerca, appariranno nello spazio sottostante i risultati inerenti alla chiave inserita.
La ricerca può essere effettuata sui campi Nome e Cognome.
