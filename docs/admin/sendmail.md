# Invia email

<img src="../../images/sections/admin/sendmail/empty.png" alt="Invia email" style="box-shadow: 0 .25rem .75rem rgba(0,0,0,.05); border: 1px solid #E9EBEE; border-radius: 0.3px;"/>

Nella menù Servizi della sezione Admin è possibile selezionare lo strumento [Invia email](/admin/sendmail).
Con questo strumento è possibile inviare un messaggio email ad uno o più utenti.

La struttura del messaggio è la seguente:

- **Oggetto (*):** L'oggetto del messaggio email.
- **Destinatari (*):** Gli utenti a cui si vuole inviare il messaggio.
- **Messaggio (*):** Il contenuto del messaggio. È possibile utilizzare la sintassi [Markdown](https://guides.github.com/features/mastering-markdown/).

<img src="../../images/sections/admin/sendmail/full.png" alt="Invia email" style="box-shadow: 0 .25rem .75rem rgba(0,0,0,.05); border: 1px solid #E9EBEE; border-radius: 0.3px;"/>

*(\*) I campi sono obbligatori.*
