# Cerca documenti

<img src="../../images/sections/documents/search.png" alt="Cerca documenti" style="box-shadow: 0 .25rem .75rem rgba(0,0,0,.05); border: 1px solid #E9EBEE; border-radius: 0.3px;"/>

Nel menù Servizi della sezione Documenti è possibile selezionare lo strumento [Cerca documenti](/documents/search).
Nella sezione [Cerca documenti](/documents/search) è possibile cercare un documento utilizzando i descrittori disponibili.

Il primo campo *(Cerca un documento)* consente di cercare in base alle parole contenute nel Titolo e nella Descrizione del documento. È possibile utilizzarlo congiuntamente agli altri descrittori.

Gli altri parametri sono descritti nella sezione **[Crea documenti](/documents/create)**.
