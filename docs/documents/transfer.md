Lo strumento [Trasferisci documenti](/documents/transfer) consente di trasferire i documenti di cui detieni i privilegi ad un altro utente registrato sulla piattaforma.

- Nel primo campo *(Aggiungi un destinatario)* è possibile cercare il nome dell'utente a cui si vuole trasferire il documento e selezionarlo.
- Nel secondo campo è possible decidere se si vogliono trasferire tutti i file in vostro possesso (non tutti quelli di cui avete i privilegi) oppure se trasferire solo alcuni documenti da voi selezionati.

<img src="../../images/sections/documents/transfer/selector.png" alt="Trasferisci documenti" style="box-shadow: 0 .25rem .75rem rgba(0,0,0,.05); border: 1px solid #E9EBEE; border-radius: 0.3px;"/>

Se viene selezionata la voce *I documenti selezionati* comparirà una maschera per scegliere quali documenti trasferire. I documenti sono selezionabili cercandoli tra le varie pagine oppure utilizzando lo strumento di ricerca. 

Una volta effettuata la scelta è possibile cliccare il pulsante *Trasferisci Documenti (:fas fa-exchange-alt:)* per portare a termine l'operazione.
